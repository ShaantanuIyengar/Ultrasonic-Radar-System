# Arduino Ultrasonic Radar System

A real-time radar platform for object detection, using Arduino Mega, HC-SR04 & SG90 with live graphical visualisation via Processing GUI. Demonstrates embedded design, hardware-software integration, sensing, and data visualisation.

## Repository Contents

- **RadarSystemArduinoCode/** – Arduino Mega firmware 
- **ProcessingCode/** – Processing visualisation Code
- **Hardware/** – Circuit schematic, parts list, assembly photos
- **Working_Video/** - A video showcasing the final product 

## Summary

This project implements a 180° radar scanner using an Arduino Mega, ultrasonic sensor, and servo. Acquired distance data is sent to a custom Processing GUI for live sweep visualisation. The system is modular, hardware-agnostic, and showcases practical skills in embedded programming, hardware integration, and real-time data presentation.

## Key Technologies

- Arduino Mega (C++/embedded firmware)
- Processing (graphical interface)
- Ultrasonic range sensing & servo control
- Serial communication & interactive data transfer

## Project Features

- Real-time hardware-software integration
- Serial communication (Arduino Mega ↔ Processing)
- Modular, maintainable codebase
- Clear technical documentation and diagrams
